Annotated data has to be inspected and validated before feeding it to AI models.
Post annotation and validation stage is crucial to ensure good quality data being passed from annotation layer to data serving layer.

## Overview
This is implementation of post annotation and validation stage comprising multiple python scripts and notebooks.

* annotations parser
* annotator performance analysis
* merge jsons files into single csv for each category

## Technologies

* python
* jupyter notebooks
* plotly-dash
